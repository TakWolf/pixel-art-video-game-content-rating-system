# 像素素材：电子游戏分级制度

各国家和地区的[「电子游戏分级制度」](https://zh.wikipedia.org/wiki/%E7%94%B5%E5%AD%90%E6%B8%B8%E6%88%8F%E5%88%86%E7%BA%A7%E5%88%B6%E5%BA%A6)标识的像素化素材。
